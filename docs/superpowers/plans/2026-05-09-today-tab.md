# Today Tab Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add a "Today" tab to Thaqalayn between Home and Explore — surfaces a rotating daily reminder verse, a Continue Reading hero card driven by reading progress, and a Du'a-of-the-day rotated from the existing daily_duas corpus.

**Architecture:** Theme-aware SwiftUI screen rendered as a new tab. New `DailyMessageProvider` service loads a bundled `daily_messages.json` and selects deterministically by `dayOfYear % count`, cached in UserDefaults. Last-read state derives from existing `ProgressManager.verseProgress` (no new sync surface). Du'a rotation uses `dayOfYear % duas.count` over the existing `DuasManager`.

**Tech Stack:** SwiftUI, Combine, `@MainActor` ObservableObjects, UserDefaults, `Calendar.islamicUmmAlQura` (via `IslamicCalendarManager`). iOS 17+. Synced-folder Xcode 16 project — files added to source folders are auto-included; no `.pbxproj` edits required.

**Project conventions to honor:**
- **No fallback logic** unless explicitly requested (CLAUDE.md). Use `fatalError` for missing bundle resources; throw or branch deliberately for user states.
- **No auto-commit.** The user commits manually between tasks. Each task ends with a "user reviews + commits" step rather than an automated `git commit`.
- **No XCTest target exists.** Verification is `xcodebuild` + `#Preview` checks + simulator smoke tests. The TDD discipline maps to: write the smallest visible slice → verify it builds → verify it renders → continue.
- **Theme adaptation:** `warmInviting` uses spec hex colours; `royalAmethyst` and `modernDark` swap to `themeManager.glassEffect`, `primaryText`, `accentColor`. Banner peach gradient is saturated in all themes.

**Design doc:** `docs/plans/2026-05-09-today-tab-design.md` — read this first for the visual spec.

```swift
//
//  DailyMessageProvider.swift
//  Thaqalayn
//
//  Loads daily_messages.json and returns today's verse deterministically.
//

import Foundation
import Combine

@MainActor
final class DailyMessageProvider: ObservableObject {
    static let shared = DailyMessageProvider()

    @Published private(set) var today: DailyMessage

    private let cacheKey = "ThaqalaynDailyMessageCache"
    private let messages: [DailyMessage]

    private init() {
        guard let url = Bundle.main.url(forResource: "daily_messages", withExtension: "json") else {
            fatalError("daily_messages.json missing from bundle")
        }
        do {
            let data = try Data(contentsOf: url)
            let decoded = try JSONDecoder().decode(DailyMessagesData.self, from: data)
            guard !decoded.messages.isEmpty else {
                fatalError("daily_messages.json must contain at least one entry")
            }
            self.messages = decoded.messages
        } catch {
            fatalError("Failed to parse daily_messages.json: \(error)")
        }

        self.today = DailyMessageProvider.resolve(messages: messages, cacheKey: cacheKey)
    }

    /// Re-evaluate today's message. Called when the app becomes active across a date boundary.
    func refreshIfDayChanged() {
        let resolved = DailyMessageProvider.resolve(messages: messages, cacheKey: cacheKey)
        if resolved.id != today.id {
            today = resolved
        }
    }

    /// Test/debug helper — peek at the next message in the rotation.
    func peekNext() -> DailyMessage {
        let nextIndex = (todayIndex() + 1) % messages.count
        return messages[nextIndex]
    }

    private func todayIndex() -> Int {
        let dayOfYear = Calendar.current.ordinality(of: .day, in: .year, for: Date()) ?? 1
        return dayOfYear % messages.count
    }

    private static func resolve(messages: [DailyMessage], cacheKey: String) -> DailyMessage {
        let dateString = Self.todayDateString()
        let cache = UserDefaults.standard.dictionary(forKey: cacheKey) as? [String: Int]

        if let cache = cache,
           let cachedDate = cache["date"] as? String,
           let cachedIndex = cache["index"],
           cachedDate == dateString,
           cachedIndex >= 0,
           cachedIndex < messages.count {
            return messages[cachedIndex]
        }

        let dayOfYear = Calendar.current.ordinality(of: .day, in: .year, for: Date()) ?? 1
        let index = dayOfYear % messages.count

        let payload: [String: Any] = ["date": dateString, "index": index]
        UserDefaults.standard.set(payload, forKey: cacheKey)

        return messages[index]
    }

    private static func todayDateString() -> String {
        let formatter = DateFormatter()
        formatter.calendar = Calendar(identifier: .gregorian)
        formatter.locale = Locale(identifier: "en_US_POSIX")
        formatter.timeZone = TimeZone.current
        formatter.dateFormat = "yyyy-MM-dd"
        return formatter.string(from: Date())
    }
}
```
